# 📁 Sistema de Cadastro em Python (CRUD)

Sistema desenvolvido em Python para gerenciamento de usuários, com funcionalidades completas de CRUD (Create, Read, Update, Delete) e persistência de dados em JSON.

---

## 🚀 Funcionalidades

* Cadastro de usuários
* Listagem de usuários
* Busca por nome
* Atualização de dados
* Remoção de usuários
* Salvamento automático em arquivo JSON

---

## 🛠️ Tecnologias utilizadas

* Python
* JSON
* Biblioteca os

---

## ▶️ Como executar

1. Clone o repositório:
   git clone https://github.com/YanFSC/sistema-de-cadastro-python

2. Acesse a pasta:
   cd sistema-de-cadastro-python

3. Execute:
   python main.py

---

## 📌 Observações

* Os dados são salvos automaticamente no arquivo `dados.json`
* Caso o arquivo não exista, ele será criado automaticamente
